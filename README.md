# 塩分濃度計算ツール

料理全体の重量と使用済みの調味料情報から現在の塩分濃度を算出し、目標濃度にするために追加すべき塩分量を求める静的Webアプリです。HTML、CSS、JavaScriptのみで構成されているため、GitHub Pagesなどのホスティングでそのまま動作します。

## 主な機能
- 総重量・目標塩分濃度の入力
- 調味料ごとの使用量と塩分%を複数行で登録（行の追加／食塩100%の追加、行のダブルクリック削除）
- 各調味料の塩分量、合計塩分量、現在の塩分濃度をリアルタイム表示
- 目標濃度に到達するために必要な食塩量、および他の調味料で補う場合の必要量を計算
- リセットボタンで初期状態へ戻す機能

## 使い方
1. `index.html` をブラウザで開きます。
2. 料理の総重量と目標塩分濃度を入力します。スープ（0.6%）・おかず（0.8%）のクイックボタンも利用できます。
3. 使用済みの調味料を行ごとに入力します。必要に応じて「行を追加」または「食塩 (100%) 行を追加」を使用してください。行は表の空き部分をダブルクリックすると削除できます。
4. 画面右側に合計塩分量、現在の塩分濃度、追加すべき食塩量、指定した塩分%の他調味料で補う場合の必要量が表示されます。
5. 「リセット」で入力内容を初期化できます。

## 計算式
- 塩分量 (g) = 重量 (g) × 塩分 (%) ÷ 100
- 合計塩分量 (g) = 各調味料の塩分量の合計
- 現在の塩分濃度 (%) = 合計塩分量 ÷ 総重量 × 100
- 追加すべき食塩 (g) = max(目標% × 総重量 ÷ 100 − 合計塩分量, 0)
- 他調味料の必要量 (g) = 追加すべき食塩 ÷ (調味料の塩分% ÷ 100)
