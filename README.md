# 料理用塩分濃度計算ツール

料理の塩分管理を手軽に行えるシンプルなWebアプリです。塩と材料の重量から現在の塩分濃度を求めたり、目標とする塩分濃度から必要な塩の量を計算できます。

## 使い方
1. `index.html` をブラウザで開きます。
2. 以下のいずれか、または両方の計算フォームを利用します。
   - **現在の塩分濃度の計算**: 塩の量と材料全体の重さを入力すると塩分濃度が表示されます。
   - **目標塩分濃度に必要な塩の量**: 目標の塩分濃度（%）と材料全体の重さを入力すると、必要な塩の量が計算されます。

どちらのフォームも結果は小数点第2位までの近似値で表示されます。

## ファイル構成
- `index.html`: アプリケーション本体
- `styles.css`: UI スタイル定義
- `script.js`: 計算ロジックとフォーム制御

## 開発メモ
追加のビルドステップは不要で、静的ホスティングやローカルファイルとしてそのまま利用できます。
